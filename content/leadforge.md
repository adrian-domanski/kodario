LeadForge is a full-stack AI SaaS application that generates LinkedIn posts using the OpenAI API. It features an ASP.NET Core backend with JWT authentication, request validation, and rate limiting, combined with a Next.js and React dashboard for generating and managing posts through a credit-based system. The application is containerized with Docker and deployed online as a live demo.
