This project involved building a modern, high-performance website for a residential investment in Józefów, Poland — Osiedle Malinowskiego. Developed with React, Next.js, TypeScript, and Tailwind CSS, the platform delivers a scalable, SEO-optimized experience with full responsive support. Interactive 3D house presentations powered by Three.js, smooth galleries using Swiper.js, and subtle animations with Framer Motion enhance user engagement. The application includes real-time segment availability, detailed specifications, multilingual support, and Headless CMS integration, providing a seamless browsing experience while effectively supporting the sales process.
