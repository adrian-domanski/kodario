TradeMonkey is a full-stack financial dashboard built with React and ASP.NET Core.
The goal of the project was to create a clean and focused financial data platform without distracting news or unnecessary content.

The application includes secure JWT authentication, REST API integration, and interactive financial charts.
It demonstrates full-stack architecture with a React frontend, ASP.NET Core backend, and SQL Server database.

This project highlights API design, authentication handling, and building scalable, production-ready applications.
